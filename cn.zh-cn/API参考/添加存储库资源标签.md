# 添加存储库资源标签<a name="createVaultTags"></a>

## 功能介绍

一个资源上最多有10个标签。 此接口为幂等接口：创建时，如果创建的标签已经存在（key相同），则覆盖。

## URI

POST /v3/\{project\_id\}/vault/\{vault\_id\}/tags

## 请求示例

```
{
	"tag":{
		"key":"key1",
		"value":"key2"
	}
}
```

## 响应示例

状态码为 400 时: 

```
{
	"error_code" : "string",
	"error_msg" : "string"
}
```
