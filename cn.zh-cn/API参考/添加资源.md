## 请求示例

```
POST https://{endpoint}/v3/f841e01fd2b14e7fa41b6ae7aa6b0594/vaults/79bd9daa-884f-4f84-b8fe-235d58cd927d/addresources

{
	"resources":[
		{
			"extra_info":{
				"exclude_volumes":[
					"bdef09bb-293f-446a-88a4-86e9f14408c4"
				]
			},
			"id":"97595625-198e-4e4d-879b-9d53f68ba551",
			"type":"OS::Nova::Server"
		}
	]
}
```
